## Основная часть

### Назначение программы
Программа автоматизирует ключевые аспекты работы склада:
*   Функция (1): Регистрация и авторизация сотрудников. 
*   Функция (2): Распределение новых заказов (Менеджер).
*   Функция (3): Назначение заказов конкретным исполнителям (Бригадир).
*   Функция (4): Пошаговая сборка товаров с проверкой ячеек (Сборщик).
*   Функция (5): Мониторинг производительности и управление перерывами.

### Условия выполнения программы
*   **Минимальный состав аппаратных средств:**
    *   Процессор Intel Core i3 / AMD Ryzen 3 или аналог.
    *   Оперативная память: 4 ГБ.
    *   Сетевое подключение (LAN/Wi-Fi).
*   **Минимальный состав программных средств:**
    *   OC Windows 10/11.
    *   Node.js v18 (для сервера).
    *   SQL Server (для хранения данных).
*   **Требования к персоналу (пользователю):**
    *   Владение персональным компьютером на базовом уровне.
    *   Знание адресной системы склада (ячейка, стеллаж).
    *   Предварительный инструктаж по работе в «SmartPicker».

### Выполнение программы
*   **Загрузка и запуск программы:**
    1.  Запустите серверную часть и убедитесь в наличии связи с БД.
    2.  Запустите клиентское приложение Electron.
    3.  На экране входа введите имя пользователя и пароль.
*   **Выполнение функций:**
    *   **Выполнение функции (1):** Перейдите в раздел «Сотрудники» для управления правами (только менеджер).
    *   **Выполнение функции (2):** В разделе «Все заказы» назначьте заказ на свободную бригаду.
    *   **Выполнение функции (3):** В разделе «Заказы бригады» перетащите заказ на свободного работника.
    *   **Выполнение функции (4):** Сборщик нажимает «Начать сборку», следует в указанную ячейку, сканирует (или отмечает) товар.
*   **Завершение работы программы:**
    *   Для выхода нажмите кнопку «Выход» в профиле.
    *   Закройте окно приложения стандартным способом.

### Сообщения оператору
*   Сообщение (1): «Ошибка авторизации» — проверьте логин или пароль.
*   Сообщение (2): «Сотрудник на перерыве» — заказ не может быть назначен.
*   Сообщение (3): «Заказ собран!» — уведомление об успешной операции.
*   Сообщение (4): «Ошибка соединения» — сервер недоступен.
