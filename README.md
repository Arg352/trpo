# SmartPicker: Система Микроменеджмента Склада

Система «SmartPicker» предназначена для автоматизации складских процессов, управления очередями заказов и мониторинга работы персонала. Проект состоит из бэкенда на **NestJS** (Prisma + SQL Server) и фронтенда на **Electron**.

---

## 🚀 Функциональные возможности
*   **Старший менеджер**: Распределение новых заказов по бригадам.
*   **Бригадир**: Назначение заказов конкретным сотрудникам, управление составом бригады.
*   **Сборщик**: Пошаговая сборка заказа, навигация по ячейкам склада (стеллаж, отдел).
*   **Аналитика**: Мониторинг KPI (строк в час, процент ошибок).

---

## 🛠 Технологический стек
*   **Бэкенд**: NestJS, Prisma ORM, Passport (JWT).
*   **База данных**: Microsoft SQL Server.
*   **Фронтенд**: Electron, Vanilla JS, CSS (Glassmorphism Design).

---

## 💻 Быстрый запуск

### 1. Требования
*   **Node.js** (v18+)
*   **SQL Server** (напр. SQL Express)

### 2. Настройка Бэкенда
1. Перейдите в папку `backend`:
   ```bash
   cd backend
   npm install
   ```
2. Создайте файл `.env` в папке `backend` и укажите `DATABASE_URL`:
   `DATABASE_URL="sqlserver://localhost:1433;database=trpo;user=sa;password=YourPassword;trustServerCertificate=true"`
3. Иннициализируйте БД:
   ```bash
   npx prisma db push
   npx prisma generate
   npm run seed
   ```
4. Запустите сервер:
   ```bash
   npm run start:dev
   ```

### 3. Настройка Фронтенда
1. Перейдите в папку `frontend`:
   ```bash
   cd frontend
   npm install
   ```
2. Запустите приложение:
   ```bash
   npm start
   ```
